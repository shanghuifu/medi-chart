# MEDI-CHART

## 概要
このアプリは患者の情報や予約を一括で管理する、いわば電子カルテの簡易版です。

## 動機
私の父が経営する治療院では患者のカルテや予約を全て紙媒体で管理しています。しかし、数千人の情報を扱うとなると、受付業務の中でのミスは起こってしまいます。
